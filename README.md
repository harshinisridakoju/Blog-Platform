# Blog-Platform
# 📝 Blog Platform (UI Demo)

A modern and visually appealing **Blog Platform UI Demo** built using **HTML, CSS, and JavaScript**. This project showcases a premium blogging website interface with responsive layouts, modern design, and interactive user interface elements.

> **Note:** This project is currently a **frontend demonstration (UI prototype)** and is **not a fully functional blogging platform**. Backend functionality and complete user features are planned for future updates.

---

## 🌐 Live Demo

**Website:** https://blog-platform-eight-green.vercel.app/

---

## 📌 Project Overview

This project demonstrates the design and user interface of a modern blogging platform. It focuses on creating an engaging user experience with a clean layout, responsive design, and attractive visual elements.

---

## ✨ Current Features

* 🎨 Modern and responsive UI
* 🏠 Landing/Home page
* 🔐 Login and Registration interface
* 📝 Blog cards and blog listing layout
* 📂 Category section
* 🔍 Search interface
* 👤 User profile UI
* 💬 Comment section UI
* 📱 Mobile responsive design
* ✨ Glassmorphism-inspired design with modern styling
